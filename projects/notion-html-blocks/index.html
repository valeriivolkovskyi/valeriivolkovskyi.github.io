While working on my personal blog, I decided to try using Notion as a CMS for my articles. Rendering raw data from the API became too difficult due to the presence of multiple nested elements and hard-to-render structures.

At that time, several libraries were available on npm that could create styled pages, React components, and even Next.js-based projects. However, these were not helpful for me since I only needed a tool to convert Notion blocks to HTML.

Therefore, I began writing a module to convert Notion data to HTML. As the code grew beyond 200 lines, I decided to move it to a separate package and make it open-source.
